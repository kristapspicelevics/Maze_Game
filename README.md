# Maze Game

Vinkārša Three.js spēlīte, kurā bumbiņa jāizved ārā no labirinta. Bumbiņu var pārvietot, izmantojot A,S,D,W taustiņus.

Spēlē ir izmantota Ammo.js fizika un taustiņu funkcionalitāte.
